# 焦点资讯开发执行计划与状态

## 1. 目标

在现有项目架构下，以“分层分模块、结构清晰、可持续扩展”为原则，完成焦点资讯模块一期底座落地：

- 分类资产底座
- Provider 抽象与 Tushare 首个实现
- 同步任务记录与可观测
- 基础查询 API
- 基础管理页面

## 2. 本轮执行范围

本轮从技术方案中的 `Phase 1` 延伸到 `Phase 2(ETL 首批落地)`：

- `Phase 1`：分类资产、provider 抽象、目录同步、页面底座
- `Phase 2`：资讯条目同步（raw + 标准表）、手动采集入口、运行统计闭环
