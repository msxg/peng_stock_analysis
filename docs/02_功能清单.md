# 功能清单

## 1. 文档定位

本文档按新架构整理系统功能，分为：

1. 应用层功能
2. 数据管理层能力
3. 数据查询服务层能力
4. 平台与支撑功能

说明：

- 应用层功能面向用户和页面
- 数据管理层与数据查询服务层面向平台能力和内部服务
- 数据访问层不作为独立业务功能清单展开，另见后续轻量设计文档

## 2. 应用层功能

### 2.1 行情数据

定位：市场数据治理控制台

功能：

- 查看本地行情数据覆盖情况
- 按数据集/市场/粒度/时间范围检查数据状态
- 查看覆盖率、缺口数、最近同步时间、新鲜度摘要
- 发起同步任务
- 发起补数任务
- 查看任务执行状态和失败原因
- 查看数据质量检查结果

### 2.2 行情监测

定位：统一行情消费看板

功能：

- 监控分类管理
- 监控标的管理
- 股票/期货统一监测
- 最新价展示
- 多粒度 K 线展示
- 数据来源、新鲜度、warning/error 展示
- 分类切换、标的搜索、异常标的快速定位

### 2.3 蓝筹模式

功能：

- 蓝筹模式今日分析
- 蓝筹模式批量分析
- 标的池配置
- 标的池管理
- 标的管理
- 分析结果展示
- 信号列表/股票列表切换
- 分析结果导出
- 分析结果保存
- 历史分析记录查询

### 2.4 行情查询

定位：临时查询和检索入口

目标功能：

- 指定股票/期货即时查询
- 多粒度 K 线查看
- 统一搜索（代码/名称/缩写）
- 查询结果来源说明
- 数据不足时的降级提示

### 2.5 市场复盘

功能：

- `GET /api/v1/market/review?region=cn|us|both`
- 输出市场概览、指数行情、板块涨跌快照

### 2.6 分析与任务

功能：

- `POST /api/v1/analysis`：发起分析（同步/异步）
- `GET /api/v1/analysis/tasks`：查询任务列表
- `GET /api/v1/analysis/tasks/:taskId`：查询任务状态
- `GET /api/v1/analysis/tasks/stream`：SSE 任务流
- 分析输出包含：
  - `technical.quote/latestCandles/volumeProfile/checklist`
  - `sentiment`
  - `fundamental_context`
  - `strategy`
  - `market_review`
  - `dashboard`

### 2.7 历史记录与回测

历史记录：

- `GET /api/v1/history`：分页查询历史
- `GET /api/v1/history/:id`：历史详情
- `GET /api/v1/history/:id/markdown`：报告 Markdown
- `GET /api/v1/history/:id/news`：关联新闻
- `DELETE /api/v1/history`：批量删除

回测：

- `POST /api/v1/backtest/run`：运行回测
- `GET /api/v1/backtest/results`：回测明细
- `GET /api/v1/backtest/overall-performance`：总体表现
- `GET /api/v1/backtest/stock-performance`：分股票表现

### 2.8 股票导入与检索辅助

功能：

- `POST /api/v1/stocks/parse-import`：文本/CSV/Excel/图片导入解析
- `POST /api/v1/stocks/extract-from-image`：图片提取
- 名称转代码
- 导入结果字段：`code/name/confidence/confidenceLevel/source`

### 2.9 Agent 问股

功能：

- `GET /api/v1/agent/models`：模型列表
- `GET /api/v1/agent/strategies`：策略列表
- `POST /api/v1/agent/chat`：问股
- `POST /api/v1/agent/chat/stream`：流式问股
- `GET /api/v1/agent/chat/sessions`：会话列表
- `GET /api/v1/agent/chat/sessions/:sessionId`：会话消息
- `DELETE /api/v1/agent/chat/sessions/:sessionId`：删除会话

### 2.10 持仓管理

功能：

- 账户：新增/查询/修改/删除
- 交易：新增/查询
- 现金流水：新增/查询
- 公司行为：新增/查询
- 快照：`GET /api/v1/portfolio/snapshot`
- 风险：`GET /api/v1/portfolio/risk-report`

### 2.11 系统配置与认证

功能：

- 认证：状态、登录、登出、开关、改密
- 配置：读取、更新、校验、Schema、LLM 渠道测试
- 邮件通知：SMTP 配置 + 测试接口 `POST /api/v1/system/test-email`
- 用量：`GET /api/v1/usage/summary`

## 3. 数据管理层能力

定位：行情数据生产与治理层

能力：

- 外部行情源接入
- 主源/备源切换与降级
- 本地行情基础表写入
- EOD 强同步
- Intraday 弱同步
- 手动补数
- 查询触发刷新
- 缺口检查
- 数据质量巡检
- 新鲜度策略维护
- 同步任务与补数任务跟踪

覆盖的数据表：

- `stock_intraday_bars`
- `stock_eod_bars`
- `futures_intraday_bars`
- `futures_eod_bars`
- `market_sync_jobs`
- `market_sync_job_items`
- `market_data_quality_reports`

## 4. 数据查询服务层能力

定位：统一查询和聚合层

能力：

- 股票/期货统一查询入口
- `intraday/eod` 路由
- `1m -> 5m/15m/30m/60m` 实时聚合
- `1d -> 1w/1M/1Y` 实时聚合
- 最新价查询
- K 线查询
- 摘要查询
- 新鲜度判断
- warning/error 标准化
- stale 数据返回与异步刷新
- missing 数据同步刷新后返回

统一输出：

- `quote`
- `series`
- `quoteSource`
- `seriesSource`
- `freshness`
- `warning`
- `error`
- `fetchedAt`

## 5. 平台与支撑功能

### 5.1 资讯与摘要

功能：

- 多资讯源管理
- 分类映射与调度
- 新闻入库、去重、聚类
- AI 摘要生成
- 市场/专题摘要沉淀

### 5.2 数据基础资料

功能：

- 股票基础资料缓存
- 期货基础资料缓存
- 名称、行业、市场、交易时段等基础信息维护

### 5.3 数据审计与追踪

功能：

- 同步任务主表与子任务记录
- 失败原因追踪
- 质量巡检报告归档
- 分析结果历史保存
